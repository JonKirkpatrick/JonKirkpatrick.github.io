A Sample of My Work

1. Build-a-Bot Stadium (BBS)

The Problem: I wanted a flexible platform to allow quick iteration on AI and ML experiments without being tied to any particular platform or stack, and to be able to keep a persistent record of my iterations.

The Approach: Built a Go-based distributed runtime using a custom JSONL-over-TCP protocol. Implemented a plugin system allowing language agnostic hot-swappable environments, remote bot instances and a real-time telemetry dashboard using Server-Sent Events (SSE). Developed a front end client interface using C# with Avalonia to have a light and easy entry point to control and monitor the action.

Go (Golang) C# (with Avalonia) TCP/IP Sockets Distributed Systems SQLite

2. Tactile Pattern Classification

The Problem: Real-time interpretation of complex tactile sensor data requires low-latency processing that exceeds the capabilities of standard software stacks.

The Approach: Developed a hardware-accelerated classification pipeline. Leveraged Verilog/HDL for gate-level signal processing paired with Python-based models for verification and pattern analysis.
